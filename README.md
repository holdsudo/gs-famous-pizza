# G's Famous Pizza — Guallpa's (Iselin, NJ)

Live one-page ordering site for Guallpa's Famous Pizza, 1522 Oak Tree Road, Iselin, NJ.
Real menu (40 items from their live Grubhub listing), real photos from their social pages,
working cart + checkout, watch-party promos, and click-to-call.

Static site — no build step. `npm start` to serve locally, `npm test` for Playwright smoke tests.
